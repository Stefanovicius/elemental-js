Elemental.js

The lightweight, no-nonsense JavaScript library for building reactive UIs with a touch of magic. Don't just write code - control the elements!

Intro

Tired of the heavyweight frameworks? Want something simple yet powerful? Elemental.js is your spell-book for creating dynamic, responsive web interfaces with minimal fuss. Simple syntax, powerful features - all in one compact package, with no external dependencies.

To Get Started with Elemental.js

Add it through a CDN:

          
            <script src="https://cdn.jsdelivr.net/npm/elemental-js@latest"></script>

Or install via npm or whatever package manager you prefer, though I really recommend you to try out Bun/Deno: 

              
                bun add elemental-js
              
            
              
                npm install elemental-js
              
            
              
                pnpm add elemental-js
              
            
              
                yarn add elemental-js

Counter Example

          
            import { el, def } from 'elemental-js'
  
            function Counter() {
              const count = def(0)
              const increment = () => ++count.val

              return el`button onclick=${increment}`(
                'Count is: ', count
              )
            }
  
            l('#counter').append(Counter())

Why

Elemental.js was born out of a desire for a lightweight, dependency-free JavaScript library that could be used to write userscripts. Also for learning and fun.
